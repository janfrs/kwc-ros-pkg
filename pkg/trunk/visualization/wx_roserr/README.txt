This package depends on wxWidgets 2.8.
On Ubuntu, install the libwxgtk2.8-dev.  On other systems, install the
appropriate package.

In order to run this package, you must also be running a botherder.  

Run make on the Makefile provided.  To use this panel as a window, run roserr_test in the bin folder.
To include this panel in a window of your own, see the roserr_test source code (test.cpp) in the src folder.
